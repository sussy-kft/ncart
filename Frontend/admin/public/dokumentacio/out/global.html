InputMezo(as, flag, input, isSelect, handleChange, value, pool, name, idk, veryCoolValue, checked) → {React.Element}

InputMezo egy komponens, ami egy input form vezérlőt jelenít meg. Az input form vezérlő az `input` segítségével állítja be a szüksges adatokat. Egy flag-et is kezel, hogy a handleChange függvényt csak egyszer hívják meg, így az adatok alapértelmezetten be lesz állítva. Továbbá lekéri opcióit a szerverről (ha esetleg nem opciókat a `pool`-ból), kezeli a változásokat és beállítja az alapértelmezett értéket. Ha a `dataType` egy tömb, akkor egy checkboxot jelenít meg. Vagy ha a `references` vagy a `isSelect` nem üres, akkor egy select form vezérlőt jelenít meg a SelectMezo segítségével.
Parameters:
Name Type Description
as string Egy komponens típusa, amibe majd beágyazza az input form vezérlőt.
flag boolean Egy flag, amit a handleChange függvényt csak egyszer hívják meg, így az adatok alapértelmezetten be lesz állítva.
input Object Az input mező metaadatai.
isSelect boolean Egy flag, amit a select vezérlő értékének beállításához használ, hogy az első elemet válassza ki vagy a kapott értéket.
handleChange function Egy callback függvény, ami a select vezérlő változásait kezeli.
value string Az input vezérlő alapértelmezett értéke. Ezt az értéket lehet módisítani az input vezérlőn keresztül.
pool Array Az input vezérlő opcióinak listája. A listában lévő elemekből választhat a felhasználó.
name string Az input vezérlő neve.
idk boolean Egy flag, amit az input vezérlő értékének beállításához használ, hogy az első elemet válassza ki vagy a kapott értéket.
veryCoolValue string Az input control értékének beállításához használt érték. Ezt az értéket nem lehet módosítani a select vezérlőn keresztül.
checked boolean Egy flag, amit a select vezérlő értékének beállításához használ, hogy a checkbox be legyen-e pipálva vagy sem.
